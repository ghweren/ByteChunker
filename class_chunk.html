Подробное описание

Класс для представления порции (чанка) данных с идентификатором.

Класс Chunk используется для хранения и обработки порций данных, которые содержат идентификатор. Данные могут быть представлены в виде строки шестнадцатеричных символов, а также восстановлены из этой строки.
