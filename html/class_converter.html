Подробное описание

Класс для конвертации объектов Chunk в строки и обратно.

Класс Converter предоставляет методы для преобразования векторов объектов Chunk в вектор строк и обратно.

Методы

◆ chunksToStrings()

static std::vector< std::string > Converter::chunksToStrings ( const std::vector< Chunk > & chunks)
inlinestatic

Преобразует вектор чанков в вектор строк.

Аргументы
chunksВектор объектов Chunk, которые нужно преобразовать в строки.
Возвращает
Вектор строк, каждая из которых представляет один чанк в шестнадцатеричном формате.

Метод chunksToStrings принимает вектор объектов Chunk и преобразует каждый чанк в строку, представляющую его данные в шестнадцатеричном формате. Результатом является вектор строк, каждая из которых соответствует одному чанку.

◆ stringsToChunks()

static std::vector< Chunk > Converter::stringsToChunks ( const std::vector< std::string > & strings)
inlinestatic

Преобразует вектор строк в вектор чанков.

Аргументы
stringsВектор строк, которые нужно преобразовать в объекты Chunk.
Возвращает
Вектор объектов Chunk, каждая из которых соответствует одной строке.

Метод stringsToChunks принимает вектор строк, где каждая строка представляет данные в шестнадцатеричном формате, и преобразует каждую строку обратно в объект Chunk. Результатом является вектор объектов Chunk.
